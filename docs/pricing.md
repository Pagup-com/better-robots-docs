# Pricing and editions

Better Robots.txt is available in **Free**, **Pro**, and **Premium**.

## Which edition is right for you?

<div class="table-wrap">

| Edition | Best for | Key idea |
|---|---|---|
| Free | Most WordPress sites starting out | Better default robots.txt + guided setup |
| Pro | Publishers, agencies, advanced governance | Deeper controls for AI, SEO tools, and policy |
| Premium | Protection-first sites | Strongest archive, bot, and stricter governance controls |

</div>

## What each edition includes

### Free

- Guided setup flow
- Essential preset
- Search engine visibility controls
- Core WordPress protection controls
- Basic bad bot protection
- Review before publish

### Pro

- Everything in Free
- AI-First preset
- Deeper AI & LLM governance controls
- More detailed SEO tool controls
- Optional `llms.txt` support
- More advanced module-level decisions

### Premium

- Everything in Pro
- Fortress preset
- Stronger archive and anti-scraping options
- Broader bad bot coverage
- The strictest protection-oriented choices
