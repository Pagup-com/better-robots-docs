# Machine-First

Better Robots.txt publishes a machine-readable layer from the root of `better-robots.com`.

## Public files

- `/.well-known/ai-governance.json`
- `/llms.txt`
- `/humans.txt`
- `/ai-manifest.json`
- `/links.json`
- `/product.jsonld`
- `/organization.jsonld`
- `/author.md`
- `/dualweb-index.md`

## Purpose

These files help machines understand:
- what this site is
- what the product is
- where the canonical plugin repository lives
- where pricing, contact, governance, and documentation are located

## Important note

These files do not replace the human-facing documentation.
They complement it.
