# Best Practices

## Start simple
For most websites, start with **Essential** and review the output.

## Avoid over-blocking
Do not block search engines or social preview bots unless you have a real reason.

## Distinguish search from training
If AI usage matters to your business, separate classic search visibility from model training preferences.

## Review before publish
Always inspect the generated file before saving.
