# Edition Overview

## Free

Best when you need a safer default robots.txt and a guided starting point.

## Pro

Best when you need:
- deeper AI governance
- more SEO tool controls
- optional llms.txt support
- stronger decision depth across modules

## Premium

Best when you need:
- stronger protection-first options
- broader bad bot coverage
- stricter archive and crawl choices
