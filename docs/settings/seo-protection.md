# SEO Tool Protection

**Location:** `Step 3 - SEO Tool Protection`

This step blocks groups of SEO analysis crawlers.

## Available Toggles

- **Block Basic SEO Tools**
  Covers the basic group, including SemrushBot and DotBot
- **Block Extra SEO Tools**
  Covers the larger crawler group, including AhrefsBot, MJ12bot, Serpstat, Screaming Frog, and others

## How to Use It

Use the two crawler-group toggles to decide how aggressively you want to block SEO analysis tools.

## Plan Notes

- SEO crawler blocking is available on paid plans
