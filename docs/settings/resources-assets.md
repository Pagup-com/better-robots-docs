# Resources & Assets

**Location:** `Step 9 - Resources & Assets`

This step controls two groups of allow rules.

## Available Toggles

- **Allow CSS & JS Files**
- **Allow Images**

## How It Works

- When CSS/JS is enabled, the plugin adds allow rules for key stylesheet and script patterns
- When images are enabled, the plugin adds allow rules for image paths
