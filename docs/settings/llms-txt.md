# LLMS.txt File

**Location:** Step 12 — LLMS.txt File

This step manages the optional virtual `/llms.txt` file.

## What it includes

- toggle to enable the file
- content field for the file body
- resulting route when enabled

## Editions

- Free: not available
- Pro: available
- Premium: available

## Important note

The file is not auto-written for you. You provide the content. Better Robots.txt then publishes it at `/llms.txt` when enabled and populated.
