# LLMS.txt File

**Location:** `Step 12 - LLMS.txt File`

This step manages an optional virtual `/llms.txt` file.

## What the Step Includes

- A Pro/Premium-only toggle to enable the virtual file
- A textarea for the file contents
- A link showing the resulting `/llms.txt` URL when enabled

## When It Becomes Available

`/llms.txt` becomes available when all of the following are true:

- You are on `Pro` or `Premium`
- `Enable llms.txt Virtual File` is turned on
- The `LLMS.txt Content` field is not empty

If the content field is empty, the route exists in code but no file content is served.

## Important Note

You write the `llms.txt` content yourself in this step.

## Suggested Use

Use this step if you want a dedicated machine-readable file that explains how AI systems should interact with your site.
