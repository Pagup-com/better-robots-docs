# Global Settings (Sitemaps)

**Location:** Settings ➔ Global Settings

This module controls the foundation of your sitemap integration, allowing you to tell search engines exactly where to find your content.

## Sitemap Integration

By default, any detected sitemaps are automatically appended to the end of your `robots.txt` file.

**Benefits:**

- **Automatic sitemap discovery** for search engines
- **Better search engine indexing** speed
- **No manual configuration needed** for supported plugins

### Supported Sources

Better Robots.txt automatically detects sitemaps generated by:

- ✅ **WordPress Native** - The default WordPress sitemap functionality
- ✅ **Yoast SEO** - Full auto-detection for Yoast sitemaps
- ✅ **Rank Math** - Complete compatibility with Rank Math
- ✅ **All in One SEO** - Native integration and optimization
- ✅ **SEOPress** - Complete compatibility

### Custom Sitemaps

You can manually provide up to 5 custom sitemap URLs. This is particularly useful if:

- You use a custom sitemap generator
- You have external sitemaps (e.g. `https://my-domain.com/static-sitemap.xml`)
- You are running a complex multisite architecture

**How to add a custom sitemap:**

1. Navigate to **Global Settings > Global Sitemaps Locations**.
2. Click **Add Sitemap URL**.
3. Enter the absolute URL (e.g., `https://example.com/sitemap.xml`).
4. **Save Changes**.

## Generated Output Example

```txt
# Sitemap
Sitemap: https://yourdomain.com/wp-sitemap.xml
Sitemap: https://yourdomain.com/custom-sitemap.xml
```
