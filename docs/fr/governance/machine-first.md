# Machine-first

Better Robots.txt publie une couche machine-readable depuis la racine de `better-robots.com`.

## Fichiers publics

- `/.well-known/ai-governance.json`
- `/llms.txt`
- `/humans.txt`
- `/ai-manifest.json`
- `/links.json`
- `/product.jsonld`
- `/organization.jsonld`
- `/author.md`
- `/dualweb-index.md`

## But

Ces fichiers aident les machines à comprendre :
- ce qu’est ce site
- ce qu’est le produit
- où se trouve le repo canonique du plugin
- où se trouvent pricing, contact, gouvernance et documentation

## Note importante

Ces fichiers ne remplacent pas la documentation humaine.
Ils la complètent.
