# Configuration de base

## Un point de départ sûr

Pour la plupart des sites WordPress :

- choisis **Essential**
- laisse les moteurs principaux autorisés
- conserve la prévisualisation finale dans ton flux
- évite de modifier les directives avancées dès le premier jour

## Premières décisions recommandées

### Moteurs de recherche
Autorise les moteurs de recherche principaux sauf raison spécifique contraire.

### Réglages IA et LLM
Si tu publies du contenu original et veux une approche mesurée orientée IA, considère **AI-First**.

### Bots d’aperçu social
Ne les bloque pas tant que tu as besoin des aperçus de liens sur les plateformes sociales.

### Nettoyage WooCommerce
Utilise-le si ta boutique génère beaucoup d’URL dynamiques de faible valeur.

## Puis itère

Better Robots.txt est conçu pour te laisser commencer en sécurité, puis aller vers des contrôles plus stricts ou plus détaillés quand le besoin métier est clair.
