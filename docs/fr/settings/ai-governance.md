# Gouvernance IA & LLM

**Emplacement :** `Étape 2 - Gouvernance IA & LLM`

Cette étape contrôle la manière dont les systèmes IA sont gérés dans les règles robots de votre site.

## Contrôles principaux

- **AI Training Protection**
  Bloque le groupe de robots d’entraînement lorsqu’il est activé
- **AI Search & Answer Engines**
  Permet de choisir entre :
  - `Block All AI Search`
  - `Allow AI Search Engines`

Le fichier séparé `/llms.txt` est documenté dans [Fichier LLMS.txt](/fr/settings/llms-txt).
